# Goal

This application helps creating projects, setting goals for each one of them and then add actions to achieve them.

## Planned features

- [x] CRUD projects API
- [x] CRUD goals API
- [x] CRUD actions API
- [ ] CRUD projects UI
- [ ] CRUD goals UI
- [ ] CRUD actions UI
- [ ] Reorder actions by priority
- [ ] Assign deadlines for each action
